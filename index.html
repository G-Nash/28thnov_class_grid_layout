Grid layout:

Grid can modify the contents on both directions i.e, both x and y.

  • display:grid
  • fr = occupies the empty space
  • grid-template-column
  • grid-template-column: repeat(4,1fr):
    • 4 mean 4 columns, 1fr mean each gets 1fr space
    • can give all the column values in 1 line
    • repeat (auto-fill, minmax(300px 1fr))
      • 300px is min value and 1fr is max value
  • gird-template-rows:
    • can give row height
